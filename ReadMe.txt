# LRS Web Application Project For APDEV Course

The Computer Lab Reservation System is designed to efficiently manage student computer lab bookings. 
It offers features such as reservation management, user authentication, and lab availability tracking. 
Users can register, login, and seamlessly reserve lab slots, while administrators have access to additional functionalities. 

Development Tools/Language
Editor: VS Code, MongoDB Compass
Frontend: React, 
Backend: Node.js 
Database: MongoDB and AWS S3
