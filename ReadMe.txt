## Getting Started

After cloning the repository, follow these steps to start development:

1. Open the terminal in VS Code.
2. Navigate to CCAPDEV_MCO.
3. npm install, then npm run dev

//it will run a script to npm run dev for both API AND Client workspace at once.
    

Always execute these commands when making code modifications to visualize the changes through the provided port.

## Using Git in Terminal

Ensure Git is installed before using these commands.

### Stage Files for Commit

To stage files for committing:

```bash

git add {filename}  # Add specific file

or

git add .  # Add all changes

Commit Changes

Commit your changes with a descriptive message:

    git commit -m "message here - context of what has been modified"

Publish Changes to GitHub
Publish your changes to the GitHub repository:

    git push
